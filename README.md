# CLEVER
Cluster-Level Eviction for Vector Embedding Retrieval

‼️ To run the project follow the following steps:
1. Clone the repository: `git clone https://github.com/your_username/CLEVER.git`
2. Navigate to the project directory: `cd CLEVER`
3. Create a virtual environment: `python -m venv .venv`
4. Activate the virtual environment: `source .venv/bin/activate` (for Unix/Linux) or `.venv\Scripts\activate` (for Windows)
5. Install the required dependencies: `pip install -r requirements.txt`
6. Make sure that you have the Hugging Face Token API for the dataset.
7. Run the file `src/data_loader.py`. This will help you get the data and store it in the `data` directory.
8. Run the file `embeddings/miniLM_embeddings.py`. This will generate the embeddings and store them in the `data` directory.
